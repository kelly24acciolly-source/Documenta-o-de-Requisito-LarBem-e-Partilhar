Sistema de Apoio à Instituição Partilhar

Sobre o Projeto

Este projeto foi desenvolvido na disciplina de Computação, Sociedade e Sustentabilidade, com o objetivo de criar uma solução digital acessível para a Instituição Partilhar.
A proposta integra ferramentas gratuitas para melhorar a organização da instituição e ampliar seu impacto social.

Objetivo
Facilitar doações
Organizar voluntários
Registrar atendimentos
Aumentar visibilidade digital
